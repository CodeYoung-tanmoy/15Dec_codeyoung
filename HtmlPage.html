Headings

There are 6 types of headings in HTML and they are given below.

Codes

<h1> Heading Text-1 </h1>
<h2> Heading Text-2 </h2>
<h3> Heading Text-3 </h3>
<h4> Heading Text-4 </h4>
<h5> Heading Text-5 </h5>
<h6> Heading Text-6 </h6>

Output

Heading Text-1

Heading Text-2

Heading Text-3

Heading Text-4

Heading Text-5
Heading Text-6

Paragraph

Codes

<p> This is a paragraph. </p>

Output

This is a paragraph.

Bold Text

Codes

<b> This will make the things bold. </b>

Output

This will make the things bold.

Italics Text

Codes

<i> This will make the things italics. </i>

Output

This will make the things italics.

Underlined Text

Codes

<u> This will underline the text. </u>

Output

This will underline the text.
